# Authentication System

A modern Authentication System built using FastAPI, HTML, CSS, SQLAlchemy, and SQLite database.

This project allows users to:
- Register Account
- Login Securely
- Store User Data
- Authenticate Users
- Use Password Hashing
- Access Dashboard After Login

The backend is built using FastAPI and SQLite database while the frontend is developed using HTML and CSS.
